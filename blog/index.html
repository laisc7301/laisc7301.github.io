Docker入门教程

Posted on 2025-03-28 In 编程 , Docker Waline:

1. Docker 的基本概念

  1. 镜像（Image）：
    • Docker 镜像是运行容器的基础，比如 ubuntu、nginx、mysql 等。
  2. 容器（Container）：
    • 运行中的镜像就是容器，容器是一个独立的环境，类似于虚拟机，但比虚拟机更轻量。
  3. 仓库（Registry）：
    • 存放 Docker 镜像的地方，比如 Docker Hub（官方公共仓库），你也可以搭建私有仓库。
